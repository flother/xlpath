See also
========

- Excel's XLSX file format is one of the Office Open XML file formats, standardised as
  [ECMA-376](https://ecma-international.org/publications-and-standards/standards/ecma-376/). Part 1,
  Fundamentals and Markup Language Reference, contains all the gory details on the available XML
  elements and attributes.

- XPath expressions are CSS selectors on steroids. To find out what's possible, see the
  [XPath cheatsheet on devhints.io](https://devhints.io/xpath).
